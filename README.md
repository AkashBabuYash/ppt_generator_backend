# PPT Generator Backend

Live Website: https://aitechpptmaker.netlify.app/

## Overview

AI Tech PPT Maker is an AI-powered presentation generator that creates professional PowerPoint presentations automatically from user prompts. Users can enter any topic, and the platform generates structured slides with titles and content using AI.

This repository contains the backend server responsible for:
- AI content generation
- PPT generation logic
- API handling
- Database management
- Frontend integration

---

## Features

- AI-powered presentation generation
- Automatic slide creation
- REST API support
- Fast backend processing
- MongoDB Atlas database integration
- Frontend and backend integration
- Scalable architecture
